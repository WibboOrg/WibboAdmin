# Wibbo CMS - Frontend

Bienvenue dans le CMS Frontend de Wibbo, un système de gestion de contenu basé sur Nuxt.js 3. Ce CMS vous permet de créer, éditer et gérer facilement le contenu de votre site Web Wibbo.

## Prérequis

Avant de commencer, assurez-vous d'avoir installé Node.js et NPM sur votre système.

## Installation

1. Clonez ce dépôt sur votre machine locale:
```bash
git clone https://github.com/votre-utilisateur/wibbo-cms-frontend.git
```

2. Accédez au répertoire du projet:
```bash
cd wibbo-cms-frontend
```

3. Installez les dépendances:
```bash
npm install
```

## Configuration

1. Renommez le fichier `.env.example` en `.env`:

2. Modifiez le fichier `.env` pour configurer votre environnement.

## Utilisation

1. Lancez l'application en mode développement:
```bash
npm run dev
```

2. Accédez à l'interface utilisateur du CMS dans votre navigateur:
```http
http://localhost:3000
```
